Function assert_eq

    Function std::assert::assert_eq

    pub fn assert_eq<T>(v1: T, v2: T)
where
    T: Eq
    Expand description

    Asserts that the given values v1 & v2 will always be equal during runtime.

    Arguments

    • v1 - The first value to compare.
    • v2 - The second value to compare.

    Reverts

    Reverts when v1 != v2.

    Examples

    fn foo(a: u64, b: u64) {
    assert_eq(a, b);
    // if code execution continues, that means `a` is equal to `b`
    log("a is equal to b");
}
